
====================Install目录安装包使用方法====================
1、安装后需要自己将配置文件放到云通讯共享文件下面。
2、配置文件首次运行应用会自己生成名字为config.plist，使用ifunBox或者itunes打开手机，找到文件夹云通讯平台的共享目录或显示为Docment目录，将里面的config.plist文件复制出来修改为自己的账号信息后再复制回去覆盖原文件，然后重启应用。
